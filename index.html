🔒 SITE_A 專屬綁定設定

此頁面為 SITE_A 專屬版本。設定將以 SITE_A_v1 前綴儲存，與其他網站完全隔離。

📋 設定步驟

  1. 前往 Firebase Console
  2. 建立新專案或選擇現有專案
  3. 啟用 Authentication → Google 登入方式
  4. 建立 Realtime Database
  5. 前往「專案設定」→「一般」→「您的應用程式」，複製設定填入下方
  6. （選填）貼上 Make.com Webhook URL 一併綁定

▌ Firebase Config

▌ Make.com Webhook URL（選填）

儲存後作為 SITE_A 的專屬 Webhook 組件。所有發送請求都將硬編碼 site_id: "SITE_A"。

⚠️ 提示：設定完成後自動重新載入頁面。所有設定以 SITE_A_v1_ 為前綴儲存於瀏覽器，與其他網站的 localStorage 完全隔離。
